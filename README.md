# Medicine Delivery, Blood Test & Blood Donation Management System

A simple healthcare web app offering medicine delivery, blood test
booking, and blood donation features. Users can browse and order
medicines, schedule lab tests, register as donors, or request blood,
making essential healthcare faster and more accessible.

## Features

-   Online medicine ordering
-   Blood test scheduling
-   Blood donation and request system
-   Clean and responsive interface

## Technologies Used

-   HTML
-   CSS
-   JavaScript

## How to Run

1.  Clone the repository.
2.  Open `index.html` in your browser.
